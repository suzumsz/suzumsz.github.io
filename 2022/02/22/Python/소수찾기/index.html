Python) 소수 찾기

난이도 : Level 2
유형 : 완전탐색
카테고리 : 프로그래머스
날짜 : 2022-02-22

문제 풀이

  1. 사용되어야 할 변수

  2. 입력받은 문자열 numbers의 한자리 수부터 len(numbers)의 수까지 순열(permutations)로 만들어준다.

  3. 순열로 만들어 준 수를 join 연산자를 사용하여 합쳐준다. 예) (‘1’, ‘7’) -> 17

  4. 각각의 수가 소수인지 검사하여, 소수면 리스트 answer에 추가해준다.
