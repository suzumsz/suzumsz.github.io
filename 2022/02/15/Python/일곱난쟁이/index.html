문제 풀이

sum(arr) - (arr[i] + arr[j]) == 100이 되면 해당 난쟁이를 새로운 변수로 넣어두고 마지막에 arr에서 제거한다.
sum(arr) - (arr[i] + arr[j]) == 100이 되면 반복문을 한번 더 선언하고 해당 난쟁이를 제외하고 출력 후 exit()로 빠져나온다.
break문을 사용하게 되면 해당 반복문만 빠져나오고 다시 반복문을 돌기 때문에 주의한다.
